# Avis

<strong>Avis</strong> is a simple country based News client.

- Browse between headlines or categorized news for more than 50 country
- Dark theme support
- Bookmark your favourite news

## Techs Stack

- React Native
- TypeScript
- Expo
- React Navigation
- React Query

## API Services

NewsAPI (https://newsapi.org)
